# Rock Paper Scissor
Players must correctly choose between "Stone," "Paper," or "Scissors" to beat their opponents. The standard rules apply:

<li>Stone crushes Scissors.</li>
<li>Scissors cuts Paper.</li>
<li>Paper covers Stone.</li>
